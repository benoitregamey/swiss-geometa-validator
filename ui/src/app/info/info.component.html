The swiss geometa validator provides a standalone tool in form of a web application and an API to validate XML files encoded in ISO19139.che against the swiss geospatial metadata profile.

The ISO19139.che is the XML implementation of the swiss geospatial metadata profile.

The ISO19139.che profile is used in the swiss geodata catalogue geocat.ch. Usually, all (geo)metadata in geocat.ch must follow the profile. The swiss geometa validator provides a handy tool to validate the content and structure of metadata before importing or harvesting them into geocat.ch.
